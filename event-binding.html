Event Binding

Event binding is a relatively simple process that causes a certain type of event (click, mouseover, mouseout, mousedown etc.) to run a certain set of instructions. This method of binding functions is also known as unobtrustive JavaScript, as the event call is not placed into the HTML tag. For example:

<button onclick='doTheThing()'>Do the thing</button> - This is obtrusive, and simply adds the onclick function to this button.

<button>Do the thing</button>
<script>
$(function(){
$('button').click(function(){
//do stuff
});
});
</script>
The above code would bind the function to all button elements.
